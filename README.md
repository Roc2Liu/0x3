# 🔍 0x3 - 搜索引擎聚合导航

> 一个美观、现代、完全无障碍的搜索引擎聚合导航网站。支持多引擎切换、自定义添加、主题切换、数据导入导出等功能。

## ✨ 核心特性

### 🔄 搜索引擎管理
- **多引擎切换**：快速切换不同的搜索引擎
- **自定义添加**：添加任意搜索引擎，支持 Emoji 图标或 SVG 图片
- **在线编辑**：随时编辑搜索引擎的名称、图标和 URL
- **智能排序**：支持按名称或添加时间排序
- **搜索过滤**：快速查找已添加的搜索引擎

### 💾 数据管理
- **本地存储**：所有数据存储在浏览器 LocalStorage，完全离线可用
- **导入导出**：支持加密的搜索引擎配置导入导出
- **数据安全**：基于域名和时间戳的动态加密，确保数据安全

### 🎨 主题系统
- **多种模式**：浅色、深色、自动三种主题模式
- **时间自动切换**：自动模式下，晚上 8 点到早上 6 点自动切换深色模式
- **系统跟随**：自动模式跟随系统偏好设置
- **平滑过渡**：主题切换带有平滑动画效果

### 📱 响应式设计
- **完美适配**：桌面端和移动端都有优化的布局
- **触摸友好**：移动端优化的触摸交互
- **自适应网格**：搜索引擎卡片自动适应屏幕宽度

### 📲 PWA 支持
- **可安装应用**：支持安装到桌面和移动设备
- **离线使用**：Service Worker 缓存，支持离线访问
- **自动更新**：应用自动检查并更新到最新版本
- **原生体验**：全屏显示，类似原生应用体验

### ♿ 无障碍支持
- **WCAG 2.1 AA 级别**：符合国际无障碍标准
- **键盘导航**：所有功能完全支持键盘操作
  - `Tab` / `Shift+Tab` 导航
  - `Enter` / `Space` 触发操作
  - `Esc` 关闭弹窗和下拉菜单
  - `↑` / `↓` 在列表中导航
  - `Home` / `End` 跳转到首尾
- **屏幕阅读器**：完整的 ARIA 标签和语义化 HTML
- **焦点管理**：智能的焦点陷阱和焦点恢复
- **高对比度**：支持系统高对比度模式
- **减少动画**：尊重用户的减少动画偏好

## 🚀 快速开始

### 前置要求

- **Node.js** 16.0 或更高版本
- **npm** 或 **yarn** 包管理器

### 安装步骤

1. **克隆项目**（或下载源码）

```bash
git clone <repository-url>
cd 0x3
```

2. **安装依赖**

```bash
npm install
```

3. **启动开发服务器**

```bash
npm run dev
```

开发服务器会在 `http://localhost:3000` 启动，支持热模块替换（HMR）。

4. **构建生产版本**

```bash
npm run build
```

构建的文件将输出到 `dist/` 目录。

5. **预览生产版本**

```bash
npm run preview
```

## 📖 使用指南

### 基础搜索

1. 在搜索框中输入关键词
2. 点击搜索按钮或按 `Enter` 键
3. 搜索结果会在新标签页中打开

### 切换搜索引擎

**方法：下拉菜单**
- 点击搜索框左侧的引擎选择器
- 从列表中选择目标搜索引擎
- 支持键盘导航：`↑` / `↓` 选择，`Enter` 确认

### 添加搜索引擎

1. 点击右上角的 ⚙️ **设置** 按钮
2. 在"添加搜索引擎"表单中填写：
   - **名称**：搜索引擎的显示名称（如：Google）
   - **图标**：输入 Emoji（如：🌐）或点击上传按钮选择 SVG 图片
   - **搜索 URL**：使用 `{query}` 作为搜索关键词占位符
     - 示例：`https://www.google.com/search?q={query}`
3. 点击"添加"按钮完成添加

**URL 格式示例：**
```
Google:      https://www.google.com/search?q={query}
百度:        https://www.baidu.com/s?wd={query}
Bing:        https://www.bing.com/search?q={query}
GitHub:      https://github.com/search?q={query}
YouTube:     https://www.youtube.com/results?search_query={query}
DuckDuckGo:  https://duckduckgo.com/?q={query}
```

### 编辑搜索引擎

1. 进入设置页面
2. 在"已添加的搜索引擎"列表中找到要编辑的引擎
3. 点击"编辑"按钮
4. 修改名称、图标或 URL
5. 点击"保存"或"取消"

### 删除搜索引擎

1. 进入设置页面
2. 在"已添加的搜索引擎"列表中找到要删除的引擎
3. 点击"删除"按钮
4. 确认删除操作

> ⚠️ **注意**：Bing 是默认搜索引擎，不能删除。

### 导入导出

#### 导出配置

1. 进入设置页面
2. 点击"导出"按钮
3. 浏览器会自动下载加密的配置文件（`.json` 格式）

#### 导入配置

1. 进入设置页面
2. 点击"导入"按钮
3. 选择之前导出的配置文件
4. 确认导入操作
5. 导入成功后，相同 ID 的引擎会被覆盖

> 🔒 **安全说明**：配置文件使用基于域名和时间戳的动态加密，只有本网站可以正确解析。

### 安装 PWA 应用

#### 桌面端（Chrome/Edge）

## 💡 常见问题

### Q: 如何添加其他搜索引擎？

A: 进入设置页面，找到要添加的搜索引擎的搜索 URL 格式，将搜索关键词替换为 `{query}`，然后填写名称、图标和 URL 即可。

### Q: 数据存储在哪里？

A: 所有数据（搜索引擎列表、当前引擎、主题设置）都存储在浏览器的 LocalStorage 中，完全离线可用，不会上传到服务器。

### Q: 导出的配置文件可以跨设备使用吗？

A: 可以。导出的配置文件可以在任何设备上导入，但需要注意加密是基于域名和时间的，确保在相同域名下使用。

### Q: 支持哪些图标格式？

A: 支持两种图标格式：
1. **Emoji**：直接输入 Emoji 字符（如：🔍、🌐）
2. **SVG 图片**：通过上传按钮选择 SVG 格式的图片文件

### Q: 如何删除所有自定义搜索引擎？

A: 目前需要逐个删除。或者可以清除浏览器的 LocalStorage，然后重新加载页面（注意：这也会清除其他设置）。

### Q: 搜索引擎卡片布局不正常怎么办？

A: 项目使用响应式网格布局，会自动适应不同屏幕尺寸。如果遇到布局问题，可以尝试：
1. 刷新页面
2. 清除浏览器缓存
3. 检查浏览器是否支持 CSS Grid

### Q: 键盘导航不工作？

A: 确保：
1. 没有其他扩展程序干扰
2. 浏览器支持键盘导航
3. 尝试使用 `Tab` 键而不是鼠标
